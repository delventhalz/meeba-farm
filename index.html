Meeba Farm

Meeba Farm is an idea I've been kicking around since I ran into Rednuht's awesome Genetic Cars. It got me excited to build my own evolving simulator with actual feeding, breeding, and dying. This is version one. Watch as the meeba's bounce around and drain calories from each other using feeding spikes. Each one has their own unique genome that they will pass on (with some mutations) once they've eaten enough to reproduce. See which lineage's succeed and which die out, click to add new meebas to the tank, or mess with the settings below and see what changes.
